# Ecommerce Frontend Design

A modern, responsive ecommerce website frontend built with React, Vite, and TailwindCSS.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Product Catalog**: Browse products by categories (Books, Clothing, Tech, Interior, Services)
- **Shopping Cart**: Add items to cart and manage quantities
- **Product Details**: View detailed information about products
- **Search and Navigation**: Easy navigation with sidebar and header components
- **Modern UI**: Clean, professional design with TailwindCSS styling

## Tech Stack

- **React** - Frontend framework
- **Vite** - Build tool and development server
- **TailwindCSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **React Icons** - Icon library
- **ESLint** - Code linting

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MuhammadZaighamAsif/ecommerce-frontend-design.git
   cd ecommerce-frontend-design
   ```

2. Navigate to the project directory:
   ```bash
   cd ecommerceSite
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint for code quality checks
